# HumanChain - AI Safety Incident Dashboard

![HumanChain Logo](public/logo_small.png)

A modern, responsive dashboard for tracking and managing AI safety incidents. Built with React, TypeScript, and a focus on user experience.

## 🌟 Features

- **Real-time Incident Tracking**: Monitor AI safety incidents with detailed information
- **Severity-based Filtering**: Filter incidents by severity level (Low, Medium, High)
- **Advanced Search**: Search through incidents by title and description
- **Sorting Options**: Sort incidents by date (newest/oldest)
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Dark Theme**: Eye-friendly dark mode interface

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: CSS3 with CSS Variables
- **Routing**: React Router
- **State Management**: React Hooks
- **Font**: Inter (Google Fonts)

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/umesh-2003/human-chain.git
cd human-chain
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 🎨 Design Decisions

- **Color Scheme**: Dark theme with blue accents for better readability and reduced eye strain
- **Typography**: Inter font for clean, professional appearance
- **Layout**: Responsive grid system for optimal viewing on all devices
- **Animations**: Subtle transitions and hover effects for better user feedback
- **Accessibility**: Semantic HTML and ARIA labels for better screen reader support
